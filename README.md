# Sistema Empresa e Clientes

Projeto desenvolvido em **JavaScript utilizando Programação Orientada a Objetos (POO)**.

O objetivo do projeto é simular uma empresa que possui clientes cadastrados, onde cada cliente possui um endereço e pode possuir múltiplos telefones.

# Como executar o projeto

1️⃣ Pré-requisitos
Ter o Node.js instalado na máquina.

No terminal, dentro da pasta do projeto, execute:

node nome-do-arquivo.js
